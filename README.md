# Probability Distribution Visualizer

This project demonstrates an interactive visualization of probability distributions of functions like Beta, Chi-square and Poisson using [stdlib](https://stdlib.io/) distribution functions.

---

## Installation and Project Setup

1) Cloning the repository:-
```bash
git clone <Your_forked_repo_link>
```

2) Installing dependencies:-

```bash
npm install
```

3) Building using Webpack & Running it locally:-

```bash
npm run build
npm run serve
```

---

Tools used
- [stdlib.js](https://github.com/stdlib-js/stdlib) (`@stdlib/stats-base-dists-*`)
- [D3.js](https://d3js.org/)
- [noUiSlider](https://refreshless.com/nouislider/)
